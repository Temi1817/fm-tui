# fm-tui — Terminal File Manager

> Двухпанельный файловый менеджер для терминала, вдохновлённый Far Manager и Total Commander.
> Написан на Python + [Textual](https://textual.textualize.io/).

## Что это такое?

**fm-tui** — быстрый файловый менеджер, который работает прямо в терминале.  
Не нужен GUI, не нужен X11 — запускается везде, где есть Python.

**Для кого:** разработчики, DevOps, все кто любит работать в терминале.

---

## Возможности

| Категория | Что умеет |
|-----------|-----------|
| **Панели** | Два независимых панели, навигация клавишами |
| **Превью** | Подсветка синтаксиса, метаданные, превью изображений |
| **Редактор** | Встроенный редактор прямо в панели превью (F4) |
| **Командная строка** | Выполнение shell-команд с выводом внутри fm (`:`) |
| **Архивы** | Просмотр и навигация внутри .zip / .tar — как обычная папка |
| **Поиск** | Быстрый поиск `/`, глубокий поиск по подпапкам `Ctrl+F` |
| **Операции** | Копирование, перемещение, переименование, удаление в корзину |
| **Выделение** | Множественное выделение `Space`, выделить всё `Ctrl+A` |
| **Закладки** | Сохранить папку `b`, открыть список `'` |
| **Иконки** | Nerd Font иконки по типу файла (опционально) |
| **Платформы** | Windows, macOS, Linux |

## Установка

### Через pipx (рекомендуется)

```bash
pipx install fm-tui
```

### Через pip

```bash
pip install fm-tui
```

> Все зависимости (включая Pillow для превью изображений) устанавливаются автоматически.

### Удаление

```bash
pipx uninstall fm-tui
# или
pip uninstall fm-tui
```

## Запуск

```bash
fm                   # открыть в текущей папке
fm /path/to/dir      # открыть в указанной папке
fm --icons           # включить Nerd Font иконки
fm --version
fm --help
```

## Клавиши

### Навигация

| Клавиша | Действие |
|---------|----------|
| `↑` / `↓` или `k` / `j` | Перемещение курсора |
| `Tab` | Переключить активную панель |
| `Enter` | Открыть папку / файл / архив |
| `Backspace` | Выйти на уровень выше |
| `Alt+←` / `Alt+→` | История назад / вперёд |

### Файловые операции

| Клавиша | Действие |
|---------|----------|
| `F5` | Копировать в другую панель |
| `F6` | Переместить в другую панель |
| `F2` | Переименовать |
| `F7` | Создать папку |
| `F8` / `Del` | Удалить (в корзину) |
| `F3` | Просмотр файла в панели превью |
| `F4` | Редактировать файл (встроенный редактор) |
| `Ctrl+D` | Дублировать файл в той же папке |
| `Ctrl+R` | Массовое переименование выделенных файлов |

### Архивы

| Клавиша | Действие |
|---------|----------|
| `Enter` на архиве | Войти внутрь архива (.zip, .tar, .gz…) |
| `Ctrl+E` | Извлечь архив в текущую папку |
| `Ctrl+Z` | Создать .zip из выделенных файлов |

### Встроенный редактор (F4)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+S` | Сохранить файл |
| `Esc` | Отменить редактирование |

### Командная строка

| Клавиша | Действие |
|---------|----------|
| `:` | Открыть командную строку |
| `: <команда>` | Выполнить любую shell-команду |
| `: git status` | Пример: вывод команды отображается прямо в fm |
| `: help` | Встроенная справка |
| `: q` | Выход |

### Выделение

| Клавиша | Действие |
|---------|----------|
| `Space` | Выделить / снять выделение |
| `Ctrl+A` | Выделить всё / снять всё |

### Поиск и вид

| Клавиша | Действие |
|---------|----------|
| `/` | Поиск в текущей папке |
| `Ctrl+F` | Глубокий поиск по всем подпапкам |
| `Esc` | Закрыть поиск |
| `.` | Показать / скрыть скрытые файлы |
| `s` | Сортировка: имя → размер → дата |

### Прочее

| Клавиша | Действие |
|---------|----------|
| `b` | Добавить текущую папку в закладки |
| `'` | Открыть список закладок |
| `!` | Открыть терминал в текущей папке |
| `?` | Справка |
| `q` | Выход |

## Конфигурация

Файл настроек создаётся автоматически при первом запуске:
- **Linux / macOS:** `~/.config/fm/config.toml`
- **Windows:** `%APPDATA%\fm\config.toml`

```toml
theme = "dark"
show_hidden = false
sort_by = "name"      # name | size | date
sort_reverse = false
icons = false

bookmarks = [
    "/home/user/Projects",
    "/home/user/Downloads",
]
```
